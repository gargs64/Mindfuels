# Mindfuels

Mindfuels is a front-end demo for an e-commerce platform offering educational materials such as storybooks, activity workbooks, practice kits, and more for young learners. Bring home the latest in kids' learning materials!

## Features

- **Responsive Design**: Built to provide a seamless browsing experience across devices.
- **Product Carousels**: Interactive, scrollable components for bestselling items and featured collections.
- **Category Browsing**: Shop by age (2-4, 5-7, 8-10, 11-13) or by subjects like story books, activity books, and worksheets.
- **Dynamic Elements**: UI ready for cart manipulation, wishlist actions, and product quick-views.

## Pages

- `index.html`: The main landing page with featured items, shop by age, and testimonials.
- `products.html`: Detailed product browsing page.
- `login.html`: User authentication page.

## Technologies Used

- HTML5
- CSS3 (Vanilla)
- Vanilla JavaScript
